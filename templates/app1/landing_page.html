Welcome to the Building Permit’s Prediction Application!!


This is a graduation project done by 3 industrial engineering students at the king Abdulaziz university.

The main objective of this web application is to help the employees of Jeddah’s municipality to predict the prices of the building permits in the most accurate way.

The data collection method was done by taking data from the municipality and processing it using machine learning model that predicts the building permit fees based on the customers’ inputs.

The accuracy of the building permit predictor ≈ %90, which will provide the employee with accurate data in a more streamlined and easier way.
